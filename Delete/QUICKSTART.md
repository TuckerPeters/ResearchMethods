## What the Code Does

### Python (`fred.py`)
- Fetches 17 economic series from FRED API
- Merges them at native frequencies
- Saves as CSV

### Stata (`PetersFinal_StudentStyle.do`)
- Loads CSV and collapses to annual
- Loads GSS and aggregates by year
- Merges both datasets
- Creates 4 new variables
- Adds labels
- Makes 6 graphs
- Calculates summary statistics
- Saves final dataset

**From FRED:**
- Unemployment rate

## Common Issues & Fixes

**"CSV file not found"**
- Run `python3 fred.py` first
- Check you're in the right directory

**"GSS file not found"**
- Should be `gss7224_r2.dta` in your working directory
- If not, do-file will still work but GSS variables will be missing

**Graph exports fail**
- Make sure you have write permissions in your working directory
- Check the working directory is correct: `pwd` in terminal

**Memory error**
- In Stata: `set memory 2g` before running do-file
